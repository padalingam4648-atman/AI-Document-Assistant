# Local-RAG

A simple local RAG project structure for ingesting documents, building embeddings, and answering queries.

## Structure

- `data/` - document sources
- `db/` - vector store or database files
- `src/` - application code
- `app.py` - entrypoint script

## Setup

1. Create a Python virtual environment in `.venv`
2. Install dependencies from `requirements.txt`
3. Add your `OPENAI_API_KEY` to `.env`
